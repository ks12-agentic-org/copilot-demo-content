# 🤖 Agent Builder — Basic (ohne PayGo)

> ⏱ 6 min — Kein PayGo, keine M365 Copilot Lizenz nötig
> Access: microsoft365.com/chat → Agents → "Build an agent"

```demo
💡 Agent Builder ist für ALLE Copilot Chat Nutzer verfügbar — ohne Extra-Lizenz, ohne PayGo.
Einschränkung: Als Knowledge Source steht nur Web Content (öffentliche Websuche) zur Verfügung.
Keine Datei-Uploads, kein SharePoint, keine Firmen-Daten.

Trotzdem beeindruckend: In 5 Minuten einen spezialisierten Web-Recherche-Agenten bauen.
```

## Step 1 — Agent Builder öffnen

```demo
microsoft365.com/chat → linke Sidebar → "+" oder "Agents" → "Build an agent"
ODER: Teams → Copilot → "+" → "Build an agent"
```

## Step 2 — Agent beschreiben

```demo
Name und Beschreibung in der Conversational UI eingeben. Copilot hilft beim Formulieren.
```

### Agent definieren (Spracheingabe / Describe-Tab)

```prompt
I want to build an agent that helps our sales team research potential customers before a meeting. The agent should search the web for: company overview, recent news, financial signals, key decision makers, and relevant industry trends. It should give structured summaries — ready to use in 2 minutes before a call. Professional tone, always cite sources.
```

## Step 3 — Conversation Starters hinzufügen

```prompt
Add these conversation starters:
• "Research [Company Name] before my meeting tomorrow"
• "What's in the news about [Company] this week?"
• "Give me a competitive overview of [Industry] for my pitch"
• "Who are the key decision makers at [Company]?"
```

## Step 4 — Agent testen

```demo
Knowledge Source: nur Web (öffentliche Inhalte, kein SharePoint, kein Firmen-Datei-Upload)
Trotzdem: sofort einsatzbereit für externe Recherche.
```

### Test: Kundenrecherche vor Meeting

```prompt
Research Fabrikam AG before my meeting tomorrow. Give me: company overview (size, HQ, business), recent news from the last 4 weeks, their key strategic initiatives, and 3 conversation starters I can use in an introductory meeting.
```

### Test: Markttrend-Recherche

```prompt
Give me a structured overview of the current state of industrial automation in European manufacturing for 2025-2026. Focus on: key trends, technology drivers, top vendors, and what decision makers are currently prioritizing. Keep it under 300 words.
```

## Step 5 — Agent teilen

```demo
Teilen via Link (org-intern). Jeder im Unternehmen mit dem Link kann den Agent nutzen.
Kein Admin-Approval nötig für Personal/Team-Agents.

💬 Say: "Ohne einen Euro extra ausgegeben, ohne IT-Ticket: ein spezialisierter 
Recherche-Assistent für das Sales-Team. 5 Minuten Aufwand.
Was er nicht kann: eigene Firmendaten einbinden. Dafür braucht es PayGo oder M365 Copilot."
```

## Was Basic (ohne PayGo) kann und nicht kann

```demo
✅ KANN (Agent Builder ohne PayGo):
• Agent Builder vollständig nutzbar
• Benutzerdefinierte Anweisungen + Conversation Starters
• Web Search als Knowledge Source (öffentliches Internet)
• Agent via Link teilen (org-intern)
• Persönliche und Team-Agenten

❌ NICHT MÖGLICH (braucht PayGo oder M365 Copilot):
• Datei-Upload als Knowledge Source
• SharePoint / OneDrive anbinden
• Eigene Firmen-Daten im Agent
• Tenant-Daten (Mails, Meetings, Calendar)
• SAP / Salesforce via Graph Connectors
• Actions (automatisch E-Mails senden, Datensätze erstellen)

→ NÄCHSTER SCHRITT: Mit M365 Copilot Lizenz — alles davon freischalten.
```
