```demo
Faster, sharper responses across all M365 Copilot experiences. Noticeably quicker — especially on long document summarization and complex prompts.
```

```demo
Liquid glass UI, text formatting in prompts, cleaner chat view with better citations. Streamlined app menu — easier navigation for all Copilot experiences on the go.
```

## 🧠 Claude Model in Word April 2026

```demo
Users can now choose Claude (Anthropic) as the AI model for drafting and editing in Word. Better for long-form writing, technical specs, and documents requiring precise language — ideal for technical engineering documents.
```

## 📄 Notebooks → Word Documents May 2026

```demo
Generate Word documents directly from Copilot Notebooks content. Specify document type, audience, themes — Copilot creates a first draft ready to edit in Word.
```

## 📋 Plan Mode in Excel May 2026

```demo
Before changing anything, Copilot shows you a step-by-step plan of what it intends to do. Review, adjust, then confirm. Perfect for complex production KPI workbooks where accidental changes are costly.
```

## 🐍 Python Support in Excel May 2026

```demo
Copilot can now apply Python-powered analysis — advanced data transformations, visualizations, multi-step calculations. No Python knowledge required. Copilot writes and runs the code.
```

## 🎨 Image Model Choice April 2026

```demo
Choose between GPT-Image, Flux, or Auto for slide image generation. Different models = different visual styles. GPT-Image for photorealistic, Flux for conceptual/artistic.
```

## 🌐 Web Grounding in PowerPoint April 2026

```demo
Paste a URL as a source when creating slides — Copilot uses the web content to inform slide content. Great for product pages, news articles, public reports as slide sources.
```

## 📓 Notebooks → PowerPoint May 2026

```demo
Generate a full presentation from Copilot Notebooks. Choose focus, level of detail, slide count, design theme. Copilot pulls from notes and references to build a structured, editable deck.
```

```demo
Next-generation image generation in PowerPoint with improved detail, accuracy, and style control. More photorealistic results for product imagery, manufacturing visuals, and presentation graphics.
```
