## Create Deck from Sales Pipeline Doc (0:55–0:59)

```demo
Start from the Q2 Sales Pipeline doc — Copilot generates a full board presentation with structure, content, and slides. This is the highest-value demo for account managers.
```

### PowerPoint → Copilot → Create presentation from file → attach doc → paste prompt.

```prompt
Create a presentation from the Q2 Sales Pipeline document [upload your own pipeline file or use the provided Q2_Sales_Pipeline.docx sample]. I need a 6-slide board deck: 1) Executive Summary with total pipeline and key highlights, 2) Top opportunities table (top 3 deals), 3) Risk analysis — the highest-risk deal needs its own slide, 4) Q3 targets and what we need to close, 5) Recommended actions for the board, 6) Appendix with full pipeline table. Professional tone. No fluff.
```
